PORTUGUÊS
-> Esse programa da a possibilidade ao usuario de colar uma frase padrão para uma chave pré definida, 
por exemplo caso vc queira definidir a chave ACEITO para a frase NEGOCIAÇÃO ACEITA, M&M AGRADECE A SUA PREFERÊNCIA, 
o usuario poderá executar o programa passando a chave como parametro e ela sera automaticamente copiada para o clipboard.

PARA ABRIR UM PROGRAMA NO CMD ESCREVA CS E COLOQUE O CAMINHO DA PASTA
TAMBÉM É PRECISO INSTALAR O PYPERCLIP PARA RODAR ESSE PROGRAMA

ENGLISH
-> This program create a automatic clipboard based on some Keyword, for example if the user want to write '''Yes, I agree. That sounds fine to me'''
for the '''agree''' keyword he/she can do it automatically running the code and passing the key as an argument.
